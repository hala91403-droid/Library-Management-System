# Library-Management-System.
A Java-based Library Management System developed as a team project to apply Object-Oriented Programming concepts.

Description:
This project simulates a simple library system where users can manage books, members, and borrowing operations using predefined sample data.

Features:
Display books and library members.
Add new books and members.
Search for books by different information.
Manage borrowing and returning operations.
View recent borrowing activities.
Generate information about: 
Late book returns.
Recently borrowed books.
Members who borrowed books within a specific period.
Borrowed books and their authors.

Technologies Used:
Java
Object-Oriented Programming (OOP).

Note;
The project uses predefined books and members to demonstrate the system functionality.
